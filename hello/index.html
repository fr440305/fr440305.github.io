24点计算器

  • 用来计算 24 点。仓库里有 C 和 Web App 的实现。
  • “24点”是一种数字游戏。首先，玩家随机从一副扑克牌中抽取四个数字（J 算作 11，Q 算作 12，K 算作 13）。抽取完毕后，玩家需要只通过加减乘除，用这四个字算出 24，每个数字都要用到，且只能用一次。最后，玩家要回答计算方法。例如，玩家抽到了 7、7、4、5 四个数字，则答案应为：7 * 5 - 7 - 4。
  • GitHub 仓库 - https://github.com/fr440305/pt24

统计参数计算器

  • Web App。用来算 1D 数据集（就是一堆数字）的统计参数，比如平均数、中位数、众数、方差、标准差等等。
  • GitHub 仓库 - https://github.com/fr440305/statcalc
